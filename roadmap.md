## ESP32 Communication

| Method    | Use                         |
| --------- | --------------------------- |
| HTTP      | Send data to website/API    |
| MQTT      | IoT messaging (best method) |
| WebSocket | Real-time control           |
| BLE       | Phone communication         |
| ESP-NOW   | ESP to ESP communication    |
| OTA       | Wireless firmware updates   |
